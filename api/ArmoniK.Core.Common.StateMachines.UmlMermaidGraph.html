Format(StateMachineInfo)

Generate a UML Mermaid graph from the state machine info The current implementation will output a string whose last three lines do not correspond to mermaid code and have to be taken out explicitly. If the output string (OutputString) is to be used in a markdown document, it's also necessary to enclose it in proper mermaid markers. e.g., mermaid OutputString
